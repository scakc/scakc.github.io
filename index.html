Aerodynamic Acoustics
Prediction of Acoustic Field for Flow past Reflexed Shape Aero foil.
Aerodynamic Acoustics

This is my B.tech Project in which our aim was to analyse the source of sound of an aerodynamic wing vibrating due to open flow over it. Purspose of finding the source properties was so that it can be reduced and damped using other techniques. I have used matlab to generate the simulation data using Computaional Fluid Dynamics. The Navier's Strokes and Wave Equation were solved simultaneously to obtain the results. Grid was generated using orthogonal grid generation method.
